HOW TO CREATE TEST IN CLASSMARKER?

STEP 1. Go to the Tests section.

STEP 2. Select 'New Test'.

STEP 3. Enter a Test name and select 'Start Adding Questions'.

STEP 4. Congratulations! You have created your first Test. You can now start adding Questions.
